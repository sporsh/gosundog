A physically based path tracer written in Go.

## Features / Plans
- [x] Depth of field
- [x] Spheres
- [x] Planes
- [ ] Importance sampling
- [ ] Incremental rendering
- [ ] Acceleration structures
- [ ] Signed distance fields
- [ ] Spectral rendering
- [ ] Motion blur
- [ ] Object instancing
- [ ] Animation
